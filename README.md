## MOVIE TICKET BOOKING SYSTEM 

A console based movie ticket booking system developed using C language. This application allows users to view loaded movies, book tickets, cancel tickets, view booking history through a menu driven interface. 

## FEATURES 
1. Display available movies
2. Book Tickets 
3. View seats
4. Cancel Tickets 
5. Booking History 
6. Final print of ticket

## CONCEPTS USED 
1. Structures
2. Arrays 
3. Strings 
4. Functions
5. Conditional statements 
6. Loops
7. Standard input/output 

## HOW TO RUN THE CODE 
1. Clone the repository or download source code
2. Open terminal in the project directory 
3. Compile the program 
``` gcc Movie_ticket_booking_system.c.c
```./a.exe
